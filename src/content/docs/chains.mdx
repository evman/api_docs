## Amount Formats

Each chain uses different units for raw amounts:

| Chain | Unit | Conversion |
|-------|------|------------|
| ETH/BNB/MATIC | wei | 1 = 10^18 wei |
| BTC/LTC | satoshi | 1 = 10^8 satoshi |
| USDT/USDC (ERC-20) | micro | 1 = 10^6 micro |
| TRX | sun | 1 TRX = 10^6 sun |
| SOL | lamports | 1 SOL = 10^9 lamports |
| XRP | drops | 1 XRP = 10^6 drops |
| XMR | piconero | 1 XMR = 10^12 piconero |

API responses include both human-readable `amount` and raw `amount_raw` fields.
