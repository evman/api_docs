## Quick Overview

The Wallet API is a standalone payment service for detecting cryptocurrency deposits. It handles HD wallet management, address generation, and payment detection with confirmation tracking.

  <Card title="Multi-Chain Support" icon="rocket">

  <Card title="Static Addresses" icon="document">

## Base URL

```
https://api.yourdomain.com/v1
```

All API requests require authentication via Bearer token.

---

## Quick Start

```bash
# 1. Create a wallet
curl -X POST https://api.yourdomain.com/v1/wallets \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"chain": "eth"}'

# 2. Generate address for a user
curl -X POST https://api.yourdomain.com/v1/addresses \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"chain": "eth", "external_id": "user_123"}'

# 3. Check for payments
curl https://api.yourdomain.com/v1/payments?external_id=user_123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```
