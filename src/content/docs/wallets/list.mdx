---
title: List Wallets
description: Retrieve all configured wallets
---

<span class="http-method get">GET</span>
<span class="endpoint-path">/wallets</span>

Get a list of all configured HD wallets.

---

## Example Request

```bash
curl https://api.yourdomain.com/api/v1/wallets \
  -H "X-API-Key: sk_live_abc123..."
```

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `wallet_id` | string | Unique wallet identifier |
| `chain` | string | Blockchain code |
| `name` | string | Wallet name |
| `status` | string | `active`, `inactive`, or `error` |
| `address_count` | integer | Total addresses generated |
| `created_at` | datetime | Creation timestamp |
