---
title: Get Wallet
description: Retrieve details for a specific wallet
---

<span class="http-method get">GET</span>
<span class="endpoint-path">/wallets/&#123;chain&#125;</span>

Get detailed information about a specific wallet by chain code.

---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `chain` | string | Chain code (e.g., `eth`, `sol`, `xmr`, `usdt_eth`) |

---

## Example Request

```bash
curl https://api.yourdomain.com/api/v1/wallets/eth \
  -H "X-API-Key: sk_live_abc123..."
```

---

## Response

```json
{
  "wallet_id": "wal_eth_a1b2c3d4",
  "chain": "eth",
  "name": "Main ETH Wallet",
  "status": "active",
  "next_index": 1542,
  "created_at": "2024-01-15T10:30:00Z"
}
```

---

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `wallet_id` | string | Unique wallet identifier |
| `chain` | string | Blockchain code |
| `name` | string | Wallet name |
| `status` | string | Wallet status (`active`) |
| `next_index` | integer | Next HD derivation index |
| `created_at` | datetime | Creation timestamp (ISO 8601) |
